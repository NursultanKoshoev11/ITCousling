# IT Tech

Современный одностраничный сайт IT-компании: разработка сайтов, приложений, игр, автоматизация, интеграции и инфраструктура.

## Запуск

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Форма заявки

В текущей версии форма формирует и копирует текст заявки в буфер обмена. Для автоматической отправки подключите Telegram Bot API, email-сервис или CRM endpoint в `src/App.jsx`.
